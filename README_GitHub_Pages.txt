GitHub Pages upload steps

1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. Make sure the homepage file is named index.html.
4. In GitHub, open Settings > Pages.
5. Under Build and deployment, choose Deploy from a branch.
6. Select the main branch and the /(root) folder, then save.
7. GitHub will publish the site and provide a public URL.

To personalize further:
- replace placeholder professional links on contact.html
- add Google Scholar, ORCID, GitHub, LinkedIn, and official faculty page URLs
- optionally add a headshot image file and link it from index.html
